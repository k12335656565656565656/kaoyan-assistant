# 问答模式

## 强制规则

1. 将回答内容整理为 Q&A 问答对，每个知识点一题
2. 每题格式：
   **Q1**：[问题]
   **A1**：[答案，允许多行分段]
   ---
3. 每个问题生成 3-5 张问答对
4. 答案可分段解释，用空行隔开段落
5. 禁止表格、禁止代码块、禁止 Markdown 标题

## 参考格式

**Q1**：什么是导数？

**A1**：导数是函数在某一点的变化率，几何意义是该点切线的斜率。

定义：f'(x₀) = lim_{Δx→0} (f(x₀+Δx) - f(x₀))/Δx

---

**Q2**：可导与连续的关系？

**A2**：可导必然连续，连续不一定可导。例如 y=|x| 在 x=0 处连续但不可导。
