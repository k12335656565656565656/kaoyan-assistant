# 纯要点模式

## 强制规则

1. **只输出 bullet 列表**，不输出任何段落文字
2. 要点数量不超过 5 条
3. 每条要点不超过 20 个字
4. 每条要点以 `-` 开头
5. 禁止表格、禁止标题、禁止代码块、禁止 LaTeX 公式块

## 参考格式

- 要点一（不超过20字）
- 要点二
- 要点三

> 最多5条
