## 联合分布函数

$$F(x, y) = P(X \leq x, Y \leq y)$$

## 离散型联合分布

$$P(X = x_i, Y = y_j) = p_{ij}$$

满足 $\sum_i \sum_j p_{ij} = 1$。

## 连续型联合分布

$$F(x, y) = \int_{-\infty}^x \int_{-\infty}^y f(s, t) dt ds$$

其中 $f(x, y) \geq 0$，$\int_{-\infty}^{+\infty} \int_{-\infty}^{+\infty} f(x, y) dx dy = 1$。

## 常见二维分布

### 二维正态分布 $N(\mu_1, \mu_2, \sigma_1^2, \sigma_2^2, \rho)$
$$f(x, y) = \frac{1}{2\pi \sigma_1 \sigma_2 \sqrt{1-\rho^2}} \exp\left(-\frac{1}{2(1-\rho^2)}[\frac{(x-\mu_1)^2}{\sigma_1^2} - 2\rho\frac{(x-\mu_1)(y-\mu_2)}{\sigma_1\sigma_2} + \frac{(y-\mu_2)^2}{\sigma_2^2}]\right)$$

## 典型例题

**例题1**：设 $(X, Y)$ 的联合密度 $f(x,y) = 2$（$0 < x < y < 1$），求 $P(X + Y < 1)$

**解**：$P(X + Y < 1) = \int_0^{0.5} dx \int_x^{1-x} 2 dy = \int_0^{0.5} 2(1-2x) dx = [2x - 2x^2]_0^{0.5} = 0.5$
