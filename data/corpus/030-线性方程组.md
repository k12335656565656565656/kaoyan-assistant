## 非齐次线性方程组

$$Ax = b$$

### 解的判定
- 有解 $\Leftrightarrow r(A) = r(A, b)$
- 唯一解 $\Leftrightarrow r(A) = r(A, b) = n$
- 无穷解 $\Leftrightarrow r(A) = r(A, b) < n$

## 齐次线性方程组

$$Ax = 0$$

### 解的判定
- 仅有零解 $\Leftrightarrow r(A) = n$
- 有非零解 $\Leftrightarrow r(A) < n$

## 解的结构

### 齐次方程组通解
$$x = C_1 \xi_1 + C_2 \xi_2 + \cdots + C_{n-r} \xi_{n-r}$$

其中 $\xi_1, \ldots, \xi_{n-r}$ 是基础解系，$n-r$ 为基础解系中向量个数。

### 非齐次方程组通解
$$x = x_0 + C_1 \xi_1 + \cdots + C_{n-r} \xi_{n-r}$$

其中 $x_0$ 是特解，$\xi_1, \ldots, \xi_{n-r}$ 是对应齐次方程的基础解系。

## 典型例题

**例题1**：求解 $\begin{cases} x_1 + x_2 + x_3 = 1 \\ x_1 - x_2 + x_3 = 1 \end{cases}$

**解**：$r(A) = 2$，$n = 3$，有无穷解。令 $x_3 = t$，解得 $x_1 = 1 - t$，$x_2 = 0$，通解为 $(1, 0, 0)^T + t(-1, 0, 1)^T$
