## 可分离变量方程

$$\frac{dy}{dx} = f(x)g(y)$$

**解法**：分离变量后两边积分
$$\int \frac{dy}{g(y)} = \int f(x) dx$$

## 齐次方程

$$\frac{dy}{dx} = f\left(\frac{y}{x}\right)$$

**解法**：令 $u = \frac{y}{x$，则 $y = ux$，$dy = u dx + x du$，代入后化为可分离变量方程。

## 一阶线性方程

$$y' + P(x)y = Q(x)$$

**通解公式**：
$$y = e^{-\int P(x) dx} \left( \int Q(x) e^{\int P(x) dx} dx + C \right)$$

## 典型例题

**例题1**：求 $y' + y = x$ 的通解

**解**：$P(x) = 1$，$Q(x) = x$
$$y = e^{-x} \left( \int x e^x dx + C \right) = e^{-x} (xe^x - e^x + C) = x - 1 + Ce^{-x}$$
