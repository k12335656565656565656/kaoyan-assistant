## 欧拉方程定义

形如以下形式的变系数线性微分方程称为欧拉方程：
$$x^n y^{(n)} + a_1 x^{n-1} y^{(n-1)} + \cdots + a_{n-1} x y' + a_n y = f(x)$$

其中 $a_1, a_2, \cdots, a_n$ 为常数。

## 二阶欧拉方程

最常见形式：
$$x^2 y'' + a x y' + b y = f(x)$$

## 解法：变量代换

令 $x = e^t$（或 $t = \ln x$），则
$$\frac{dy}{dx} = \frac{1}{x}\frac{dy}{dt}$$
$$\frac{d^2y}{dx^2} = \frac{1}{x^2}\left(\frac{d^2y}{dt^2} - \frac{dy}{dt}\right)$$

代入原方程后，欧拉方程化为常系数线性微分方程，可用特征方程法求解。

## 二阶齐次欧拉方程

$$x^2 y'' + a x y' + b y = 0$$

令 $x = e^t$，化特征方程：$r^2 + (a - 1) r + b = 0$。

通解形式：
- 两不等实根 $r_1 \neq r_2$：$y = C_1 x^{r_1} + C_2 x^{r_2}$
- 重根 $r_1 = r_2 = r$：$y = (C_1 + C_2 \ln x) x^r$
- 共轭复根 $r = \alpha \pm i\beta$：$y = x^\alpha [C_1 \cos(\beta \ln x) + C_2 \sin(\beta \ln x)]$

## 非齐次欧拉方程

先用变量代换化为常系数非齐次方程，再用待定系数法或常数变易法求解。
