## 分布函数法

$F_Z(z) = P(Z \leq z) = P(g(X,Y) \leq z) = \iint_{g(x,y) \leq z} f(x,y) dxdy$，则 $f_Z(z) = F_Z'(z)$。

## 公式法（特殊情形）

### 和的分布
$Z = X + Y$：
$$f_Z(z) = \int_{-\infty}^{+\infty} f(x, z-x) dx$$
若 $X$ 与 $Y$ 独立：
$$f_Z(z) = \int_{-\infty}^{+\infty} f_X(x) f_Y(z-x) dx$$（卷积公式）

### 商的分布
$Z = \frac{X}{Y}$：
$$f_Z(z) = \int_{-\infty}^{+\infty} |y| f(zy, y) dy$$

### 积的分布
$Z = XY$：
$$f_Z(z) = \int_{-\infty}^{+\infty} \frac{1}{|x|} f(x, \frac{z}{x}) dx$$

## 典型例题

**例题1**：$X \sim N(0,1)$，$Y \sim N(0,1)$ 独立，$Z = X + Y$，则 $Z \sim N(0,2)$。
