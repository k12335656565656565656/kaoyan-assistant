## y''' = f(x) 型

逐次积分得到通解：
$$y = \iint f(x) dx dx dx + C_1 x^2 + C_2 x + C_3$$

## y'' = f(x, y') 型

设 $p = y'$，则 $y'' = p'$，方程化为 $p' = f(x, p)$。

## y'' = f(y, y') 型

设 $p = y'$，则 $y'' = p \frac{dp}{dy}$，方程化为 $p \frac{dp}{dy} = f(y, p)$。

## 典型例题

**例题1**：求 $y''' = x$ 的通解

**解**：$y'' = \frac{x^2}{2} + C_1$，$y' = \frac{x^3}{6} + C_1 x + C_2$，$y = \frac{x^4}{24} + \frac{C_1}{2}x^2 + C_2 x + C_3$
