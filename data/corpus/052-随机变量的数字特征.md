## 数学期望

### 离散型
$$E(X) = \sum_i x_i p_i$$

### 连续型
$$E(X) = \int_{-\infty}^{+\infty} x f(x) dx$$

### 性质
1. $E(C) = C$
2. $E(kX) = kE(X)$
3. $E(X + Y) = E(X) + E(Y)$

## 方差

$$D(X) = E[(X - E(X))^2] = E(X^2) - [E(X)]^2$$

### 性质
1. $D(C) = 0$
2. $D(kX) = k^2 D(X)$
3. $D(X \pm Y) = D(X) + D(Y) \pm 2\operatorname{Cov}(X, Y)$

## 典型例题

**例题1**：设 $X$ 的分布律为 $P(X=0)=0.3$，$P(X=1)=0.5$，$P(X=2)=0.2$，求 $E(X)$ 和 $D(X)$。

**解**：$E(X) = 0\times0.3 + 1\times0.5 + 2\times0.2 = 0.9$
$E(X^2) = 0^2\times0.3 + 1^2\times0.5 + 2^2\times0.2 = 1.3$
$D(X) = 1.3 - 0.9^2 = 0.49$
