## 标准形式

$$y'' + py' + qy = f(x)$$

其中 $p, q$ 为常数，$f(x)$ 为已知函数。

## 解的结构

非齐次方程的通解 = 对应的齐次方程的通解 + 一个特解
$$y = y_0 + y^*$$

其中 $y_0$ 是齐次方程的通解，$y^*$ 是非齐次方程的一个特解。

## 待定系数法

### f(x) = P_n(x) 型
设特解形式为 $x^k Q_n(x)$，其中 $k$ 是特征根 0 的重数。

### f(x) = e^{\alpha x} P_n(x) 型
设特解形式为 $x^k e^{\alpha x} Q_n(x)$，其中 $k$ 是特征根 $\alpha$ 的重数。

### f(x) = e^{\alpha x}[P_m(x)\cos\beta x + P_n(x)\sin\beta x] 型
设特解形式为 $x^k e^{\alpha x}[R_l(x)\cos\beta x + S_l(x)\sin\beta x]$，其中 $l = \max(m,n)$，$k$ 是特征根 $\alpha \pm \beta i$ 的重数。

## 典型例题

**例题1**：求 $y'' - 2y' + y = x$ 的通解

**解**：齐次方程 $y'' - 2y' + y = 0$ 的通解 $y_0 = (C_1 + C_2 x)e^x$
设特解 $y^* = ax + b$，代入得 $y^* = x + 2$
通解：$y = (C_1 + C_2 x)e^x + x + 2$
