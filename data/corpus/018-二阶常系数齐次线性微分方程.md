## 标准形式

$$y'' + py' + qy = 0$$

其中 $p, q$ 为常数。

## 特征方程法

特征方程：$r^2 + pr + q = 0$

### 三种情形

1. **两个不同实根** $r_1, r_2$：
$$y = C_1 e^{r_1 x} + C_2 e^{r_2 x}$$

2. **两个相同实根** $r$：
$$y = (C_1 + C_2 x)e^{rx}$$

3. **共轭复根** $\alpha \pm \beta i$：
$$y = e^{\alpha x}(C_1 \cos\beta x + C_2 \sin\beta x)$$

## 典型例题

**例题1**：求 $y'' - 4y' + 3y = 0$ 的通解

**解**：特征方程 $r^2 - 4r + 3 = 0$，$r_1 = 1$，$r_2 = 3$
通解：$y = C_1 e^x + C_2 e^{3x}$

**例题2**：求 $y'' - 4y' + 4y = 0$ 的通解

**解**：特征方程 $(r-2)^2 = 0$，$r = 2$（重根）
通解：$y = (C_1 + C_2 x)e^{2x}$
