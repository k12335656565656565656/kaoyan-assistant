## 秩的定义

矩阵 $A$ 的秩是指 $A$ 的非零子式的最高阶数，记作 $r(A)$ 或 $\text{rank}(A)$。

## 秩的性质

1. $r(A) = r(A^T)$
2. $r(kA) = r(A)$（$k \neq 0$）
3. $r(A + B) \leq r(A) + r(B)$
4. $r(AB) \leq \min(r(A), r(B))$
5. 若 $A$ 可逆，则 $r(AB) = r(B)$，$r(BA) = r(B)$
6. 若 $A$ 与 $B$ 等价，则 $r(A) = r(B)$

## 秩与方程组

### 非齐次方程组 $Ax = b$
- 有解 $\Leftrightarrow r(A) = r(A, b)$
- 唯一解 $\Leftrightarrow r(A) = r(A, b) = n$
- 无穷解 $\Leftrightarrow r(A) = r(A, b) < n$

### 齐次方程组 $Ax = 0$
- 仅有零解 $\Leftrightarrow r(A) = n$
- 有非零解 $\Leftrightarrow r(A) < n$

## 典型例题

**例题1**：设 $A$ 为 $3 \times 4$ 矩阵，$r(A) = 2$，求齐次方程组 $Ax=0$ 的基础解系中向量个数。

**解**：基础解系向量数 $= n - r(A) = 4 - 2 = 2$
